The Labyrinth - 3D Maze

This project is a 3D maze game developed in JavaScript using the Three.js library. The goal is to provide an immersive experience navigating through a maze, with car controls, different lighting types, day and night textures, fog effects, background music, and car model switching.


Features:

3D Graphics: Maze and models rendered in 3D with Three.js.

Car Controls: Smooth movement with acceleration, deceleration, and rotation.

Dynamic Lighting: Switch between ambient light, directional light, and car headlights.

Textures: Change between day and night textures.

Visual Effects: Enable fog for added realism.

Background Music: Option to turn music on or off during gameplay.

Car Switching: Choose between different car models and colors.

Timer: Time system to challenge the player.

Interactive Menu: Intuitive interface with options to play, view tutorial, or exit.


Project Structure:

index.html: Main game page.

main.js: Game logic and controls.

styles.css: Visual styles.

Resource folders: Models, textures, images, music, and other assets.


How to Play:

Click "Play" on the main menu.

Use the buttons to switch lights, textures, fog, music, and change the car.

Navigate the maze by controlling the car and try to reach the end as quickly as possible.
